# Cynador Next.js Website

Proyecto listo para Vercel.

## Ejecutar local
```bash
npm install
npm run dev
```

## Subir a Vercel
1. Crea un repositorio en GitHub.
2. Sube esta carpeta.
3. En Vercel: Add New Project > Import Git Repository.
4. Deploy.

## Editar datos principales
- `app/page.tsx`: textos, servicios, contacto y secciones.
- `app/globals.css`: diseño visual.
